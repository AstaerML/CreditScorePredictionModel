# Модель множественной регрессии для оценки кредитного рейтинга

Модель была создана и обучена для выполнения поставленной задачи курсовой работы на тему "Реализация алгоритмов регрессионного анализа данных для оценки кредитного рейтинга". Код файла представляет собой: 
1. Импорт библиотек и загрузку датасета
2. Предобработку данных датасета с помощью методов "MinMaxScaler()" и "StandardScaler()"
3. Разделение датасета на данные для оценки и искомую переменную, создание обучающих и тренировочных выборок
4. Сощдание и обучение модели, создание предсказаний при помощи обученной модели
5. Сравнение предсказанных значений с истинными, создание таблицы и гистограммы для наглядности
6. Вывод метрик оценки точности модели (Коэффициент детерминации, MSE)
